## 개발 환경
- Unity
- C#
- Visual Studio 2022

## Convention
### Git
- **Commit 메시지 종류 설명**

|   제목   |                    내용                     |
| :------: | :-----------------------------------------: |
|   feat   |           새로운 기능에 대한 커밋           |
| refactor |          코드 리팩토링에 대한 커밋          |
|   fix    |            버그 수정에 대한 커밋            |
|  design  |      CSS 등 사용자 UI 디자인 변경 커밋      |
|   test   |                 테스트 커밋                 |
|   docs   |            문서 수정에 대한 커밋            |
|   env    | 개발환경 구축 관련 커밋(라이브러리 추가 등) |
